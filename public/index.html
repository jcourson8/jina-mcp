A Message from James

Hello there! This project is a simple Model Context Protocol (MCP) server designed to wrap the powerful Jina Reader API. It allows Language Models (LLMs) to easily fetch, read, and process content from any URL in an LLM-friendly format.

You can use the fetch_url_content tool exposed by this server to integrate web content fetching into your LLM applications. It supports various options for fine-grained control, similar to the Jina Reader API itself.

If you find this project helpful or interesting, please consider giving it a star on GitHub! Your support is much appreciated.
